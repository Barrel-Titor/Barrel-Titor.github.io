《后浪》观后感

发表于 2020-05-04 更新于 2020-05-13 分类于 随笔 阅读次数： Valine：
本文字数： 1k 阅读时长 ≈ 1 分钟

今天是五四青年节，B 站和人民日报分别推出了短片《后浪》和《青春的逆行》。


初看《后浪》，不得不说，主讲人国家一级演员何冰，演讲技巧非常高超，让我激情澎湃。尤其是“人类积攒了几千年的财富，所有的知识、见识、智慧和艺术，像是专门为你们准备的礼物”一句，我非常赞同。每每坐在大学课堂里，我都会心存感激：能够在短短几年间，习得人类几百年来在某些学科上的思维精华，我无比幸运。


然而，关掉视频，越回想，越觉得有些难过。《后浪》视频画面由 B 站各大 UP 主的素材混剪而成，这些年轻人的生活很多姿多彩。他们拿得起放得下，既能刻苦学习勇敢追梦，又能吃喝玩乐享受生活。

这不是假象，但是，绝非全部的真相。

在我们国家，有人可以来一场说走就走的旅行，环游世界，但是更多的人没有坐过飞机；有人用着 iPad 和 Apple Pencil 尽享科技带来的效率提升，有的人却没有设备用来上网课，或者需要翻山越岭才能接收网络信号……


由对于《后浪》评价的两极分化，我们就可以看到，现实很骨感，社会割裂的问题愈发严重。

2019 年我国贫困地区农村居民人均可支配收入 11567 元[1]；按照 5.5 国际美元标准，中国仍有 2.2 亿贫困人口[2]。从接受教育程度来看，42% 的人最高学历仅为初中水平，接受本科及以上教育的人只占总人口的 4%[3]。

农村女童辍学问题、城市高房价、结婚压力……仍然在为生存拼搏的年轻人，被压得喘不过气来，他们并没有所谓选择的权利。我想，《后浪》单作为 B 站的宣传短片，也许没有什么问题；但若是面向全国的年轻人，这个短片的内容，还是显得过于残忍了。


与之相比，在人民日报《青春的逆行》里，我更能看到我心目中年轻人的样子——他们之中有医生，有创业者，有消防员，有解放军，有建筑工，有车间工人……他们生活并不丰富多彩，但是他们仰望星空，脚踏实地。

他们是共和国的脊梁。


最后，我由此联想到前一阵子，浑水对于瑞幸咖啡的调查报告。他们在瑞幸的门店数人头、攒小票，拍摄监控视频，调查高管……他们实事求是，不怕麻烦，下到最基层进行调研。这，才是各行各业应有的态度。我们的一切从群众中来，也必须最后到群众中去。

如果 B 站能吸取这次的教训，想必能制作出更接地气一点的《后浪》吧。
